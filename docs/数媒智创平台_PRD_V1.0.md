2. 产品概述

2.1 产品名称与定位

产品名称：数媒智创平台（https://x.sztv.com.cn）

产品定位：面向媒体行业与内容创作者的一站式AI智能内容生成平台，集成大模型对话、热点聚合、知识库管理、AIGC内容生产与AI技能市场，帮助用户提升选题、创作、配音、视频生成等全链路效率。

2.2 目标用户

媒体记者、编辑、策划人员：需要快速追踪热点、生成选题与稿件

新媒体运营：需要跨平台内容适配、标题优化、短视频生成

内容创作者：需要AI辅助的图片、视频、音频生产

技术与业务管理员：需要管理企业知识库、发布与订阅AI技能

2.3 核心价值

热点感知：聚合国内外多平台热搜，提供实时选题参考

智能创作：基于多模型AI对话与专业智能体，快速生成文本、图片、视频、音频

知识沉淀：通过知识库实现企业文档的RAG检索与智能问答

技能扩展：SkillHub支持社区化AI技能共享，满足垂直业务场景

3. 功能架构

平台采用“主站 + 独立子系统”的架构：

主站（x.sztv.com.cn）：统一工作台，承载新建任务、热点榜单、历史对话、导航入口

创作子站（geekai.sztv.com.cn）：图片生成、视频创作等AIGC生产工具

知识库子站（weknora.scms.sztv.com.cn）：企业级RAG知识库管理（WeKnora）

技能市场子站（SkillHub）：AI技能发布、搜索与订阅

5. 典型业务流程

5.1 热点选题流程

1. 用户进入“热点榜单”，切换国内/国际Tab查看实时热搜

2. 发现感兴趣话题后，点击首页对应智能体（如【国内】多平台热搜聚合分析智能体）

3. 在智能体对话中点击“开始执行任务”，平台调用热搜接口抓取数据并生成分析报告

4. 若接口异常，系统提示“网络发生波动，请稍后重试”

5.2 AIGC内容生产流程

1. 用户进入“生产工具”→“内容创作”

2. 选择图片生成或视频创作，填写提示词、选择模型与参数

3. 点击“立即生成”，任务进入任务列表

4. 生成完成后在“创作记录/你的作品”中查看、下载、删除

5.3 数字主播视频生产流程

1. 进入“数字主播”

2. 第一步选择数字人形象（实景/动态/绿幕/半身/全身）

3. 第二步选择播报音色

4. 第三步输入文案，点击生成

5. 在生成历史中下载成品视频

5.4 知识库问答流程

1. 用户进入“知识库”→“知识管理”（WeKnora iframe）

2. 上传企业文档（PDF/Word/Excel/图片/音频/Markdown）

3. 在“知识问答”中基于知识库提问

4. 系统通过RAG检索与ReAct智能体返回答案

注：当前WeKnora SSO登录存在异常，需后端排查认证链路。

6. 非功能需求

维度

需求说明

性能

热点榜单、历史对话列表应在2秒内完成首屏加载；AI生成任务支持异步排队

兼容性

主站与各子系统需保持登录态一致性，避免切换子站后主站被登出

安全性

账号密码登录+图形验证码；敏感操作（修改密码）需二次确认；Cookie/Token仅限本地使用

可用性

关键操作（生成、登录）失败时给出明确错误提示；支持明暗主题切换

扩展性

通过SkillHub与第三方工具入口持续扩展能力，保持主站架构稳定

可维护性

各子系统独立部署（geekai/weknora/SkillHub），便于分别迭代与故障隔离
